# AgroWise

## Project info

AgroWise - Where Agriculture Meets Technology. Empowering farmers with AI-powered crop advisory, weather intelligence, pest detection, and market insights.

## Getting Started

To get started with this project locally:

1.  Clone the repository.
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
